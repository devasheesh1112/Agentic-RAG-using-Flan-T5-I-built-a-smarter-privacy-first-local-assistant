# Agentic-RAG-using-Flan-T5-I-built-a-smarter-privacy-first-local-assistant
A privacy-focused, local Agentic Retrieval-Augmented Generation (RAG) system built with Python and LangChain. This project optimizes RAG pipelines by introducing an intelligent routing layer that decides when retrieval is necessary, improving efficiency and ensuring data stays entirely offline.
